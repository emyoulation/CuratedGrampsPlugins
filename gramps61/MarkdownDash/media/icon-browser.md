## SVG File Format Support

These test whether your GdkPixbuf has librsvg2 installed (required for SVG rendering):

![](gramps:icon:gramps-person:16) Person SVG (via theme)
![](gramps:icon:gramps-family:22) Family SVG (via theme)
![](gramps:icon:gramps-event:24) Event SVG (via theme)

If the icons above show correctly, SVG is working. If they appear as
`[icon-name]` placeholders, SVG support is unavailable and only raster PNGs
at the Gramps-installed sizes (16, 22, 48) will render.
